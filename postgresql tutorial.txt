Install From the Postgresql Repositories

Access https://yum.postgresql.org/repopackages.php, get Centos link https://download.postgresql.org/pub/repos/yum/reporpms/EL-7-x86_64/pgdg-redhat-repo-latest.noarch.rpm

1. wget https://download.postgresql.org/pub/repos/yum/reporpms/EL-7-x86_64/pgdg-redhat-repo-latest.noarch.rpm
2. yum install pgdg-redhat-repo-latest.noarch.rpm epel-release
3. yum update
4. yum install postgresql11-server postgresql11-contrib
5. /usr/pgsql-11/bin/postgresql11-setup initdb
6. systemctl start postgresql-11
7. (Optional to start on boot) systemctl enable postgresql-11

After install and start postgresql
Login to root user
1. su - postgres
2. psql

Create new user
1. create user [user] with encrypted password 'password' [any roles];

To login into new user, change pg_hba.conf peer to md5
1. psql -U [user] -d [database] -W
