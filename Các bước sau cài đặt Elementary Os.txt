->>>>>>>>>> GÕ tiếng việt
Install dconf editor: apt install dconf-editor.
Open the app, search for property `source` in /org/gnome/desktop/input-source
Change it with the pattern: [('xkb', 'us'), ('ibus', 'xxx')] which `xxx` is your input source like Unikey, Teni, etc. I'm stick with Teni for better experience. You can check it out Teni-github.
Logout

->>>> install from source
1. sudo apt-get install build-essential
2. tar -xzvf file.tar.gz or tar -xjvf file.tar.bz2
3. cd /file
4. xdg-open INSTALL (de xem huong dan cua file do) Thường là theo lệnh sau
5. mkdir build && cd build
6. cmake ../
7. make
8. sudo make install
make clean-all nếu build bị lỗi hoặc thiếu phải build lại
thường thì thiếu thư viện nhiều, phải lên mạng tải mới build được. tip là một số thư viện thêm -dev đằng sau là ok
